<task type="auto">
  <name>Task 2: Implement error handling with proper status codes</name>
  <files>main.py</files>
  <read_first>main.py</read_first>
  <action>
Change the exception handler in main.py to raise HTTPException(status_code=500) instead of returning ErrorResponse with 200 status code.

Current problematic code (lines 37-44):
```python
except Exception as e:
    logger.error(f"Error scraping {source} for query '{query}': {e}")
    return ErrorResponse(
        source=source,
        query=query,
        error=str(e),
        scraped_at=datetime.now(UTC)
    )
```

Replace with:
```python
except HTTPException:
    raise  # Re-raise HTTPExceptions as-is
except Exception as e:
    logger.error(f"Error scraping {source} for query '{query}': {e}")
    raise HTTPException(status_code=500, detail=f"Scraping failed: {str(e)}")
```

This ensures server errors return HTTP 500, not 200.
  </action>
  <verify>
<automated>grep -q "raise HTTPException(status_code=500" main.py && echo "FOUND" || echo "MISSING"</automated>
  </verify>
  <done>Server errors return HTTP 500, not 200</done>
</task>

<task type="auto">
  <name>Task 3: Add rate limiting to /search endpoint</name>
  <files>main.py</files>
  <read_first>main.py</read_first>
  <action>
Add SlowAPI rate limiting to main.py:

1. Add imports at top of file:
```python
from slowapi import Limiter, _rate_limit_exceeded_handler
from slowapi.util import get_remote_address
from slowapi.errors import RateLimitExceeded
```

2. Initialize limiter after app creation:
```python
app.state.limiter = Limiter(key_func=get_remote_address)
app.add_exception_handler(RateLimitExceeded, _rate_limit_exceeded_handler)
```

3. Add rate limit decorator to /search endpoint:
```python
@app.post("/search", response_model=SearchResponse, responses={400: {"model": ErrorResponse}, 429: {"model": ErrorResponse}, 500: {"model": ErrorResponse}})
@limiter.limit("10/minute")
async def search(request: Request, request_body: SearchRequest):
```

Note: The route decorator must be ABOVE the limit decorator, not below it. Pass request as first parameter.
